# Glossaire du vocabulaire arabe – Cours de Fiqh

## Vocabulaire général

| Terme                             | Définition                                                                                                        |
|-----------------------------------|-------------------------------------------------------------------------------------------------------------------|
| **3abath**                        | Frivolité ; toute distraction ou action inutile sans bénéfice religieux ni mondain                                |
| **3awra**                         | Parties du corps devant être couvertes ; zone intime dont l'exposition est interdite                              |
| **3ibâdât**                       | Actes d'adoration (prière, jeûne, etc.)                                                                           |
| **3illa**                         | Cause légale commune à deux cas (utilisée dans le raisonnement par analogie/qiyas)                                |
| **3aqidah**                       | Croyance/credo ; domaine de la théologie islamique                                                                |
| **3urf**                          | Coutume reconnue comme source juridique, à condition qu'elle ne contredise pas le Coran et la Sunna               |
| **açl**                           | Le cas original (dans le raisonnement par analogie/qiyas)                                                         |
| **ahdâth**                        | Impureté rituelle ; rupture de l'état de pureté rituelle                                                          |
| **ahkam**                         | Jugements/prescriptions légales                                                                                   |
| **ahkam al sultâniyya**           | Droit constitutionnel ; relations entre gouvernants et gouvernés                                                  |
| **ahwâl achakhsiya**              | Droit du statut personnel ; droit de la famille de la naissance à la dissolution                                  |
| **akhlaq**                        | Éthique/morale ; domaine de la bonne conduite et du caractère                                                     |
| **aymâne**                        | Serments (pluriel de yamine)                                                                                      |
| **basmala**                       | Le fait de dire « bismillah » ; invocation du nom de Dieu avant un acte                                           |
| **bismillah**                     | « Au nom d'Allah »                                                                                                |
| **bismillahi ar-rahmân ar-rahim** | « Au nom d'Allah, le Tout Miséricordieux, le Très Miséricordieux »                                                |
| **bulûgh**                        | Puberté ; une des conditions pour être mukallaf                                                                   |
| **char3ou man quablama**          | Législations des prophètes précédents (source juridique secondaire)                                               |
| **dalk**                          | Friction ; passer la main sur un membre lavé pendant l'ablution (obligatoire chez les Malikites)                  |
| **dhanniyya**                     | Conjecturale/présomptive ; décrit les jugements ouverts à l'interprétation                                        |
| **dhikr**                         | Rappel d'Allah ; invocations et prières                                                                           |
| **dirham**                        | Unité de mesure ; utilisée pour définir la taille excusable d'une impureté de sang                                |
| **dou3a**                         | Supplication/invocation à Allah                                                                                   |
| **fadâil**                        | Actes méritoires ; actes de vertu et d'excellence                                                                 |
| **fahm**                          | Compréhension (sens général/linguistique), distingué du fiqh                                                      |
| **faqih**                         | Juriste ; celui qui comprend profondément le droit islamique et peut en dériver des jugements                     |
| **far3**                          | Le cas dérivé/nouveau (dans le raisonnement par analogie/qiyas)                                                   |
| **farâ'id**                       | Actes obligatoires (pluriel de fard)                                                                              |
| **fard**                          | Acte obligatoire ; devoir religieux                                                                               |
| **fard 3ayn**                     | Obligation individuelle ; devoir personnel que chaque musulman doit accomplir                                     |
| **fard kifâi**                    | Obligation collective ; devoir accompli si suffisamment de membres de la communauté le font                       |
| **fasl**                          | Chapitre/section (d'un livre)                                                                                     |
| **fatwa**                         | Avis juridique émis par un savant qualifié                                                                        |
| **fiqh**                          | Jurisprudence islamique ; compréhension humaine de la loi divine appliquée aux actes pratiques                    |
| **ghâib / ghâit**                 | Lieu bas/creux dans le sol ; euphémisme coranique pour les selles                                                 |
| **ghâlib**                        | Ce qui est généralement le cas ; la situation prédominante/habituelle                                             |
| **ghîba**                         | Médisance ; parler d'une personne absente d'une manière qui lui déplairait                                        |
| **ghusl**                         | Grande ablution rituelle ; purification du corps entier                                                           |
| **hadd / hudûd**                  | Limite légale fixée par Allah ; peines prescrites ou limites divines (ordres et interdictions)                    |
| **hadath**                        | Impureté rituelle (immatérielle) ; état légal empêchant certains actes d'adoration                                |
| **hadith**                        | Tradition prophétique ; parole, acte ou approbation du Prophète                                                   |
| **hajj**                          | Pèlerinage à La Mecque ; un des piliers de l'Islam                                                                |
| **hamz**                          | Moquerie/dénigrement fait en face par des regards, gestes ou mimiques                                             |
| **harâm**                         | Interdit/prohibé ; acte puni s'il est commis, récompensé s'il est évité                                           |
| **hasad**                         | Envie/jalousie ; souhaiter la disparition d'un bienfait accordé à autrui                                          |
| **houkm**                         | Jugement légal ; prescription dérivée d'un texte                                                                  |
| **hukm qat3î**                    | Jugement catégorique/définitif ; certain dans sa preuve et sa signification                                       |
| **ijma3**                         | Consensus ; accord de tous les savants d'une époque donnée sur un jugement légal                                  |
| **ijtihâd**                       | Effort d'interprétation ; raisonnement juridique indépendant à partir des sources textuelles                      |
| **inqâ**                          | Nettoyage réel/approfondi (des membres pendant l'ablution)                                                        |
| **istibrâ**                       | Précaution post-miction ; s'assurer que rien ne reste dans les voies urinaires                                    |
| **istihsân**                      | Préférence juridique ; choisir un jugement pour la facilité pratique plutôt que l'analogie stricte                |
| **istislah**                      | Prise en compte de l'intérêt public comme source juridique                                                        |
| **istidlâl**                      | Dérivation des preuves légales ; processus d'extraction des jugements à partir des sources                        |
| **janaba**                        | État de grande impureté rituelle (nécessitant le ghusl)                                                           |
| **kawasib**                       | « Les acquéreurs » ; les sept membres/organes par lesquels on acquiert le bien ou le mal                          |
| **khabath**                       | Impureté matérielle/visible (urine, excréments, etc. sur le corps ou les vêtements)                               |
| **kibr**                          | Arrogance/orgueil ; rejeter la vérité et mépriser les gens                                                        |
| **lamz**                          | Moquerie/dénigrement fait dans le dos, par la langue                                                              |
| **madhî**                         | Liquide pré-séminal ; liquide fin émis par le désir (ne nécessite pas le ghusl)                                   |
| **madhhab**                       | École de jurisprudence ; méthodologie juridique d'un imam particulier                                             |
| **madmada**                       | Rinçage de la bouche avec de l'eau puis recrachement (acte recommandé du wudu)                                    |
| **mahr**                          | Dot ; don nuptial offert à l'épouse lors du mariage                                                               |
| **mahram**                        | Parent masculin avec qui le mariage est définitivement interdit (ex : père, frère)                                |
| **makrouh**                       | Déconseillé ; acte dont l'abandon est récompensé mais dont la commission n'est pas punie                          |
| **makrouh tanzihi**               | Légèrement déconseillé ; déconseillé sans obligation contraignante de l'éviter                                    |
| **makrouh tahrîmi**               | Fortement déconseillé ; déconseillé avec obligation contraignante de l'éviter                                     |
| **maslaha al mursala**            | Intérêt public non restreint ; source juridique secondaire                                                        |
| **mash**                          | Essuyage avec les mains humides (ex : sur la tête pendant l'ablution)                                             |
| **mashhûr**                       | L'opinion bien connue/prédominante au sein d'une école de jurisprudence                                           |
| **moubâh**                        | Permis ; acte qui n'est ni récompensé ni puni                                                                     |
| **moustahab**                     | Recommandé ; acte récompensé s'il est accompli mais non puni s'il est omis                                        |
| **mu3âmalât**                     | Transactions sociales ; commerce, mariage, divorce, contrats                                                      |
| **mubtil**                        | Annulateur ; ce qui rend un acte nul                                                                              |
| **mufti**                         | Celui qui émet des fatwas (avis juridiques)                                                                       |
| **mukallaf**                      | Personne légalement/religieusement responsable (doit être musulmane, pubère, saine d'esprit, informée)            |
| **mutûn**                         | Textes didactiques versifiés/condensés (poèmes utilisés pour l'enseignement)                                      |
| **muwâlât**                       | Continuité/enchaînement ininterrompu des actes d'ablution                                                         |
| **al mutlaq**                     | Absolu/pur (eau) ; eau inchangée en couleur, goût ou odeur, valide pour l'usage rituel                            |
| **najâssa**                       | Impureté (matérielle, rituelle)                                                                                   |
| **namîmah**                       | Colportage/calomnie ; rapporter des paroles pour semer la discorde entre les gens                                 |
| **nâqid / nawâqid**               | Annulateur(s) ; ce qui invalide l'ablution (singulier/pluriel)                                                    |
| **niyya**                         | Intention ; résolution intérieure requise avant les actes d'adoration                                             |
| **oussoul al fiqh**               | Fondements/principes de la jurisprudence ; méthodologie de dérivation des jugements légaux                        |
| **qat3iyya**                      | Catégorique/définitif (adjectif décrivant certains jugements)                                                     |
| **qiyâs**                         | Raisonnement par analogie ; comparer un cas nouveau à un cas existant partageant la même cause légale             |
| **quâdi**                         | Juge ; celui qui applique les jugements légaux                                                                    |
| **radd**                          | Le passage retour (de l'essuyage de la tête, de l'arrière vers l'avant) ; considéré obligatoire par les Malikites |
| **raka3ât**                       | Unités de prière                                                                                                  |
| **ribâ**                          | Usure/intérêt ; gain financier prohibé                                                                            |
| **riyâ'**                         | Ostentation ; accomplir de bonnes actions pour être vu par les autres (annule la récompense)                      |
| **sahâbi**                        | Compagnon du Prophète                                                                                             |
| **salâm**                         | Salutation de paix                                                                                                |
| **salât**                         | La prière rituelle                                                                                                |
| **salat al janaza**               | Prière funéraire ; obligation collective (fard kifai)                                                             |
| **saoum**                         | Jeûne (ex : pendant le Ramadan)                                                                                   |
| **shari3ah / sharî'a**            | Loi islamique dans sa totalité ; la révélation divine complète guidant tous les aspects de la vie                 |
| **siwâk**                         | Bâtonnet de nettoyage des dents ; acte méritoire avant l'ablution                                                 |
| **sounna / sunna**                | Tradition prophétique ; les paroles, actes et approbations du Prophète — deuxième source du droit islamique       |
| **sujud**                         | Prosternation (ex : prosternation de récitation du Coran)                                                         |
| **sum'a**                         | Recherche de la renommée/réputation par les actes religieux                                                       |
| **sunan**                         | Actes recommandés (pluriel de sunna au sens juridique)                                                            |
| **tabarak'allah**                 | « Béni soit Allah » ; dit pour éloigner le mauvais œil en admirant quelque chose                                  |
| **tahâra**                        | Purification ; qualification légale/rituelle permettant l'adoration                                               |
| **taklîf**                        | Responsabilité légale ; imposition des devoirs religieux à une personne                                           |
| **taqwâ**                         | Piété/conscience de Dieu ; la seule supériorité reconnue par Allah                                                |
| **tartîb**                        | Ordre prescrit ; séquençage des actes obligatoires dans l'ablution                                                |
| **tawaf**                         | Circumambulation de la Ka'ba ; interdit sans wudu                                                                 |
| **tawba**                         | Repentir ; retour des actes blâmables vers les actes louables                                                     |
| **'ujb**                          | Admiration de soi/vanité ; racine intérieure de l'arrogance                                                       |
| **wadhî**                         | Liquide blanc émis après la miction (invalide le wudu)                                                            |
| **wahyi**                         | Révélation divine                                                                                                 |
| **wâjib**                         | Obligatoire ; acte récompensé s'il est accompli, puni s'il est omis                                               |
| **woudou / wudu**                 | Petite ablution ; lavage rituel de membres spécifiques avant la prière                                            |
| **yamîne**                        | Serment (singulier)                                                                                               |
| **zakât**                         | Aumône obligatoire ; impôt purificateur sur la richesse                                                           |

## Termes géographiques/historiques

| Terme        | Définition                                               |
|--------------|----------------------------------------------------------|
| **Al Koufa** | Ville en Irak ; centre de l'école rationaliste du fiqh   |
| **Hijâz**    | Région en Arabie ; centre de l'école basée sur le hadith |
